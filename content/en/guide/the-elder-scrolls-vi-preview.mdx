<Section icon={Swords} title="Next-Gen Technology: Creation Engine 3">

The technical foundation of the game is being built on a further-developed branch of Bethesda's proprietary technology, tentatively referred to as Creation Engine 3. This engine upgrade builds directly upon the rendering, physics, and streaming improvements introduced in Starfield's Creation Engine 2.

The transition to a next-generation engine is essential for realizing a seamless, highly detailed fantasy world that does not rely on frequent loading screens between wilderness areas, cities, and dungeons.

| Technology Area | Skyrim (Creation Engine 1) | Starfield (Creation Engine 2) | The Elder Scrolls VI (Creation Engine 3) |
| Engine Foundation | Legacy Gamebryo-based branch | Major rendering and animation overhaul | Further optimized branch for dense fantasy worlds. |
| World Rendering | Limited distant detail, simple shadows | Volumetric lighting, advanced terrain | Richer vegetation, improved water, longer sightlines. |
| Loading & Streaming | Separate loading screens for cities | Faster storage streaming, some transitions | Seamless movement between wilderness and cities. |
| Character AI | Basic daily routines (Radiant AI) | Advanced schedules and combat roles | Deeper faction relationships and dynamic reactions. |
| Mod Support | Original Creation Kit | Creation Kit 2 with modern pipeline | Advanced data-driven structure for custom content. |

By utilizing advanced procedural generation tools alongside meticulous handcrafted design, the development team can build a massive, seamless map without sacrificing the environmental storytelling that Bethesda is famous for.
